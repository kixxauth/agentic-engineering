# Agentic Software Engineering Tools

Tools I use for agentic software engineering; mostly prompts and documentation.

**NOTE:** My primary focus (in 2026) is working on the [Kixx framework](https://www.kixx.dev), which is written in JavaScript for Node.js, Cloudflare, and Deno. So, most of what you'll find here is coupled pretty tightly to that workflow. But, I think you'll find it interesting, and maybe useful, anyway.
